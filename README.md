# PMPL_2025

## E2E
E2E Testing
- end-to-end (e2e) memverifikasi alur aplikasi dari pers pengguna
- pengujian ini melibatkan beberapa lapisan sekaligus (UI, API, DB, Layanan Eksternal).
- Tujuan: memastikan sistem berkerja secara keseluruhan, bukan hanya unit terisolasi
- why :
	- menangkap masalah integrasi 

- piramida e2e: unit test, integration test, e2e test, UI testing
- alat : playwright, cypress, selenium/Webdriver, katalon studio, maestro, manual e2e  
- observability and debugging : tangkap
